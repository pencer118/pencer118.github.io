##지킬테마 로컬 실행

>1.Access the local project
:테마 폴더 경로로 이동

$ cd User/path/IdeaProject/folder-name

>Install npm packages
>:npm 패키지 인스톨

$ npm install

>Install Ruby dependencies
>:루비 번들 인스톨

$ bundle install

>Build Jekyll
>:지킬 빌드

$ bundle exec jekyll build

>Then run Gulp
>:Gulp 실행

```html
$ gulp
```
